#### Para correr el proyecto:

*clonar el repo

*updatea dependencias
```
> composer update
```

*crear y configurar el .env

*generar el key de artisan
```
> php artisan key:generate
```

*hacer la migracion de datos
```
> php artisan migrate:refresh --seed
```
*levantar el proyecto
```
> php artisan serve
```
